# Install the .uf2 file onto the pi pico:

Push and hold the BOOTSEL button while connecting the Pi pica via USB to your PC. It should appear as mass storage device. Drag the [.uf2 file](https://github.com/hippdani/Coil-O-Mat/blob/main/software/Spulenwickelmitzaehlautomat.ino.rpipico.uf2) onto the Pi Pico. It should automatically reboot. Your done!
